# MoviWeb2

MoviWeb2 is a movie tracking web application built with Flask, SQLAlchemy and the OMDb API.

## Features

- Create and manage users
- Add movies via OMDb API
- Update movie titles
- Delete movies
- Store data with SQLite
- Deployed on PythonAnywhere

## Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML / CSS
- OMDb API
